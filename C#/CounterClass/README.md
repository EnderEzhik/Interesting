Задача: подсчет созданных экземпляров класса.
Используется публичная статическая переменная с публичным геттером и приватным сеттером.
В конструкторе класса значение переменной счетчика увеличивается на единицу.
Для получения информации о количестве созданных экземпляров класса можно использовать переменную напрямую или вызвать статический метод, который выводит значение в консоль с дополнительным форматированием.
Дополнительно в класс добавлена публичная readonly переменная, которой в конструкторе присваивается текущее значение статической переменной на момент создания экземпляра класса, определяя номер текущего экземпляра, и метод, выводящий в консоль номер экземпляра с форматированием.
